O que é HTML?

(Voltar)

HTML (Hypertext Markup Language) é uma linguagem de marcação usada para criar e estruturar o conteúdo de uma página da web. Ele fornece uma estrutura básica para a criação de páginas da web, permitindo a inclusão de texto, imagens, vídeos, links e outros tipos de conteúdo. O navegador web (browser) interpreta o arquivo html e monta a estrutura da página, é por isso que dizem que código em html não é linguagem de programação, é apenas uma linguagem de marcação.

Comunicação de Cliente e Servidor

(Voltar)

Quando acessamos um site no navegador de internet, nós somos clientes, queremos alguma informação, pode ser um vídeo, imagem, algum texto ou aplicação. Cabe então ao cliente fazer uma requisição, ou seja, pedir essas informações ao servidor que envia várias informações, dentre elas, um arquivo html que monta a estrutura página. O servidor é um computador em algum lugar do mundo com a função de armazenar os arquivos que serão enviados ao cliente quando ele fizer a requisição, dentre outras funções.

Estrutura Básica de HTML

(Voltar)

A estrutura básica de um documento HTML é composta por três elementos principais: o DOCTYPE, o head e o body. O DOCTYPE informa ao navegador qual versão do HTML está sendo usada. O head contém informações sobre a página, como o título e meta tags, e o body contém o conteúdo real da página.

DOCTYPE

O DOCTYPE é uma declaração que informa ao navegador qual versão do HTML está sendo usada no documento. Ele deve ser a primeira linha do documento e é necessário para garantir que o navegador exiba a página corretamente.

Head

A seção head de um documento HTML contém informações sobre a página, como o título, a descrição e as palavras-chave. Também pode incluir referências a arquivos CSS e JavaScript, bem como meta tags que afetam a forma como a página é indexada pelos motores de busca.

Body

A seção body contém o conteúdo real da página, incluindo texto, imagens, vídeos e outras mídias. É aqui que o conteúdo da página é definido e organizado usando tags HTML.

Entendendo Tags

(Voltar)

As tags HTML são usadas para definir o conteúdo da página e para estruturá-lo de forma adequada. As tags são cercadas por colchetes angulares ('< >') e podem incluir atributos que afetam a forma como a tag é exibida ou comporta-se. Uma tag pode ser aberta e fechada, quando fechada fica como <tag/>, algumas tags não precisam de fechamento

Atributos de Tags

Os atributos de tags são usados para modificar o comportamento ou a aparência de uma tag HTML. Eles são definidos dentro da tag e incluem um nome e um valor, separados por um sinal de igual (=). Alguns exemplos de atributos comuns incluem o href para links e o src para imagens e vídeos.

Tags de Texto

As tags de texto são usadas para definir o texto que será exibido na página. Alguns exemplos de tags de texto incluem <h1> para cabeçalhos, <p> para parágrafos. Dentre outras como escrever em italico <i>, negrito<b> até mesmo riscado <del>

Listas

As listas são usadas para exibir informações em forma de lista, como itens em uma lista de compras ou links em uma barra de navegação. Existem três tipos principais de listas em HTML: listas ordenadas (<ol>), listas não ordenadas (<ul>) e listas de definição (<dl>).

Links

Os links são usados para permitir que os usuários cliquem em um texto ou imagem para serem levados para outra página ou recurso na web. Eles são criados usando a tag <a> e o atributo href.
